# 🚀 Avaliação de Fit - Engenheiro de IA

Aplicação interativa desenvolvida em **Python + Streamlit** para avaliar o nível técnico de candidatos com base em requisitos, diferenciais e soft skills.

---

## 🎯 Objetivo

Permitir que candidatos avaliem seu nível de aderência a uma vaga de **Engenheiro de IA**, gerando um score automático e classificação.

---

## 🧠 Funcionalidades

- ✅ Avaliação de requisitos obrigatórios  
- 🚀 Avaliação de diferenciais  
- 🧠 Avaliação de soft skills  
- 📊 Score automático com barra de progresso  
- 🔥 Classificação do candidato  

---

## 🖥️ Tecnologias

- Python
- Streamlit
- Pandas
